# Summary

Integration with ostim. NPCs should be aware of player ostim scenes, and also can perform adult actions.

By default, actions are not available, will be progressively available when sex_disposal NPC property raises. This property is autocalculated
based on speech mood analysis (seducing game), and relaxing status. 

NPCs can start animations direclty, or start an idle scene. They can also change current animation based on chat.

* Will use extended NPC data to store NPC status
* sex_prompt: Prompt offered when NPC is on ostim scene
* sex_speech_style: Speech style offered when NPC is on ostim scene
